# Перевірка локального середовища

=== "Версія"

    ```shell
    go version
    ```

=== "Версія+ awk"

    ```shell
    go version | awk '{print $3}' | sed 's/go//'
    ```

=== "Версія+ cut"

    ```shell
    go version | cut -f3 -d' '| sed 's/go//'
    ```

дасть у відповідь версію `x.y.z`

- x = 1
- y >= 26
- z -- не суттєво
