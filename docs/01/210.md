# Створення першого міні-проєкту

## Створення Основної папки

=== "Relative"

    ```shell
    mkdir Projects
    cd Projects
    ```

=== "Absolute"

    ```shell
    mkdir ~/Projects
    cd ~/Projects
    ```

## Створення папки уроку

=== "Relative"

    ```shell
    mkdir helloworld
    cd helloworld
    ```

=== "Absolute"

    ```shell
    mkdir ~/Projects/helloworld
    cd ~/Projects/helloworld
    ```

### Створення робочого файлу

=== "Relative"

    ```shell
    touch main.go
    ls
    ```

=== "Absolute"

    ```shell
    touch ~/Projects/helloworld/main.go
    ls ~/Projects/helloworld
    ```

### Наповнення робочого файлу

Відкрийте проект(папку) улюбленим редактором коду

=== "Relative"

    ```shell
    code helloworld
    ```

=== "Absolute"

    ```shell
    code ~/Projects/helloworld
    ```

Відкрийте файл та додайте в нього

```go linenums="1" title="main.go"
--8<-- "examples/01_100_helloworld.go.txt"
```

Збережіть файл, або увімкніть Авто Збереження (`Auto Save`)
